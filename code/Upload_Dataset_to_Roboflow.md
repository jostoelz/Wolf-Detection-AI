# Upload your own dataset to Roboflow to augment it there
This is the <a href="https://docs.roboflow.com/developer/command-line-interface/upload-a-dataset">instruction</a> for it. It should be executed in the terminal of the folder where the dataset is located.
```
roboflow import -w wolfdetectionai-0kca3 -p augmentation-dataset-7m4lz .\YOLO
```
